# 🤖 PlacaMonitor Bot

Bot do Telegram para monitoramento de placas veiculares com IA.

## Como usar no Telegram

- **Mande uma foto** com a legenda sendo o local onde foi tirada
- O bot lê a placa automaticamente e registra
- Se a placa já foi vista antes, avisa com alerta!

### Comandos
- `/lista` — mostra todas as placas registradas
- `/placa ABC1234` — busca histórico de uma placa específica

## Como subir no Render.com (grátis)

1. Crie conta em **render.com**
2. Clique em **New > Web Service**
3. Escolha **Deploy from a Git repo** ou **Upload**
4. Suba os arquivos: `bot.py`, `requirements.txt`, `render.yaml`
5. Clique em **Deploy**
6. Pronto! Bot online 24h
